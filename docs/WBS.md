# 테트리스 게임 프로젝트 WBS (Work Breakdown Structure)

## 1. 프로젝트 개요
*   **목표**: 웹 브라우저 기반 테트리스 게임 개발
*   **기간**: 7일 (Day 1 ~ Day 7)
*   **인력 구성**:
    *   **설계자 (Designer)**: 1명 (기획, 디자인, 문서화)
    *   **개발자 1 (Dev A)**: 1명 (메인 로직, 코어 알고리즘)
    *   **개발자 2 (Dev B)**: 1명 (UI/UX 구현, 입출력 처리)

## 2. 간트 차트 (Gantt Chart)

```mermaid
gantt
    title 테트리스 개발 일정 (7일)
    dateFormat  YYYY-MM-DD
    axisFormat  Day %d
    
    section 기획 및 설계
    요구사항 분석 및 PRD       :done,    des1, 2026-01-16, 1d
    UI 디자인 및 리소스 제작   :active,  des2, 2026-01-17, 2d
    
    section 개발 환경 구축
    프로젝트 세팅 (Vite)     :crit,    dev1, 2026-01-17, 1d
    
    section 코어 로직 구현 (Dev A)
    블록 데이터 및 회전 로직   :         dev2, 2026-01-18, 2d
    충돌 처리 및 줄 삭제       :         dev3, 2026-01-20, 2d
    점수 및 난이도 로직       :         dev4, 2026-01-21, 1d

    section UI 및 기능 구현 (Dev B)
    입력 처리 및 기본 렌더링   :         dev5, 2026-01-18, 2d
    HUD (점수, Next) 구현    :         dev6, 2026-01-20, 1d
    게임 오버 및 연출         :         dev7, 2026-01-21, 1d

    section 테스트 및 배포
    통합 테스트 및 버그 수정   :         qua1, 2026-01-22, 1d
    최종 배포 및 문서화       :         qua2, 2026-01-23, 1d
```
