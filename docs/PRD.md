# 테트리스 게임 제품 요구사항 문서 (PRD)

## 1. 개요
웹 브라우저에서 실행되는 고전 아케이드 스타일의 테트리스 게임입니다.

## 2. 게임 기본 기능
### 2.1. 테트로미노 (블록)
- 총 7종류의 테트로미노 구현:
  - **I** (Cyan)
  - **O** (Yellow)
  - **T** (Purple)
  - **S** (Green)
  - **Z** (Red)
  - **J** (Blue)
  - **L** (Orange)

### 2.2. 게임 플레이 메커니즘
- **블록 생성**: 7종류의 블록 중 하나가 랜덤하게 상단에서 생성됨
- **이동 및 회전**: 사용자의 키보드 입력에 따라 블록이 좌우 이동하거나 회전함
- **낙하**:
  - 일정 시간마다 자동으로 한 칸씩 하강
  - 사용자가 아래 방향키를 누르면 빠르게 하강 (Soft Drop)
  - 스페이스바를 누르면 바닥으로 즉시 하강 (Hard Drop)
- **충돌 감지**: 블록이 바닥이나 다른 블록에 닿으면 고정됨
- **줄 삭제 및 점수**:
  - 가로 한 줄이 블록으로 모두 채워지면 해당 줄 삭제
  - 삭제된 줄 수에 따라 점수 획득
  - 삭제된 줄 위의 블록들은 아래로 이동

## 3. 조작 방법
- **← (왼쪽 화살표)**: 블록 왼쪽 이동
- **→ (오른쪽 화살표)**: 블록 오른쪽 이동
- **↓ (아래 화살표)**: 빠른 낙하 (Soft Drop)
- **↑ (위 화살표)**: 블록 회전 (시계 방향)
- **Space Bar**: 즉시 낙하 (Hard Drop)

## 4. 게임 화면 구성 (UI)
### 4.1. 메인 게임판
- 격자 크기: 가로 10칸 x 세로 20칸 (10x20 Grid)
- 게임 플레이가 이루어지는 핵심 영역

### 4.2. 정보 표시 영역 (HUD)
- **점수 표시 (Score)**: 현재 플레이어의 점수 실시간 표시
- **다음 블록 (Next Block)**: 다음에 나올 블록을 미리 보여주는 영역
- **게임 오버 (Game Over)**:
  - 블록이 게임판 맨 위까지 쌓여 더 이상 생성될 수 없을 때 표시
  - 게임 종료 메시지 및 최종 점수 출력
